# A CRM Application for Banquet Hall Booking

## 📌 Project Description
This project is developed using Salesforce CRM to manage banquet hall bookings efficiently. It automates booking validation, billing generation, and reporting.

---

## 🚀 Features
- Custom Objects (Banquet Hall, Billing, Staff)
- Master-Detail Relationship between Banquet Hall and Billing
- Formula Fields for automatic billing calculation
- Record Triggered Flow to prevent duplicate bookings
- Apex Trigger for automatic billing record creation
- Summary Reports for booking and billing details

---

## 🛠️ Tech Stack
- Salesforce CRM
- Apex
- Flow Builder
- Reports & Dashboards

---

## 🔄 Workflow

1. User creates Banquet Hall booking
2. Flow checks duplicate booking (Date + Time)
3. If valid → record saved
4. Apex Trigger creates Billing record automatically
5. Formula fields calculate total amount
6. Report displays booking and billing data

---

## 📊 Output
- Automated billing system
- Duplicate booking prevention
- Organized report generation
